HTML & CSS

Began this phase of the project by acquiring a simple HTML template along with it's corresponding CSS stylesheet.Through various adjustments to tags and references, i iteratively arrived ath the current layout of the website.

Static website, HTTPS, DNS

Utilizing the AWS Management Console, I established an S3 bucket and activated static hosting. Additionally, I procured a custom domain (Olaolu-resume.com) via Route 53. For enhanced security, I generated an SSL certificate using AWS Certificate Manager to enable HTTPS. To ensure optimal content delivery, I established a CloudFront distribution and configured custom DNS routing through Route 53 to direct to the custom DNS.

DynamoDB Database, API and Python

Source Control & CI/CD

This part introduced me to the utilization of GitHub repository and GitHub actions. To eliminate the need for manual code uploads to the S3 bucket, I successfully set up a GitHub repository and configured a workflow that automatically synced commit from local website code to GitHub repository, and subsequently to S3 bucket on AWS.
